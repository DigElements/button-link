# button-link

A Polymer Element showing a stylized button that opens a link when clicked or tapped.

### Example
```html
    <button-link
        link="http://link"
        target="_blank"
        text="Click Me"
        show>
    </button-link>
```

### Dependencies

Dependencies are installed using [Bower](http://bower.io/):

    npm install -g bower
    bower install
